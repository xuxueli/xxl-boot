

## 扩展明细

| 模块名称              | 描述                   |
|-------------------|----------------------|
| AI模块              | AI相关模块，包括：Agent管理、知识库与WorkFlow基础能力；以及 智能客服、语音合成、图片生成、视频生成 等能力； 

## 扩展功能

### 1、AI模块

- AI助手（ai/chatbot）：对话；图片分析；
- Workflow（ai/workflow）：工作流管理，支持模型能力编排；执行历史记录；
- AI生图（ai/image）：文生图、图生图；生图流程设计，集成本地模型；
- AI视频（ai/video）：视频流程设计，集成本地Vision模型；
- 知识库：知识库管理，向量数据库集成；
